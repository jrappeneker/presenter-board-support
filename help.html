Presenter Board Help

A simple tool for displaying large text on a display for presentations.

Writing Text

Type your text in the main window. This will be shown in large type on the secondary screen when the Presenter window is open.

  • Use the font size to set your preferred font size.
  • Toggle autocorrect if needed. Note that this may cause delays in the display of text on the Presenter window. If this happens, hit the space button to complete the autocomplete.
  • Select alignment (left, center, right) with the buttons at the top.

Presenter Window

  • Press ⌘↩︎ (Command-Return) or click Display
  • The Presenter window will open fullscreen on the second monitor (or on your main screen if no second display is connected)
  • Press esc to close the Presenter window
  • You can choose which font will be displayed in the Settings.

Additional Tips

  • The Presenter font can be mirrored in the main editor by enabling “Use presenter font in text editor.”
  • You can clear the editor with the Clear button.

Display Opacity Settings

In Settings, you can adjust the background opacity of the Presenter display using the Display Opacity picker. This setting controls the translucency of the text background:

  • Choose from system materials from Ultra Thin (very transparent) to Ultra Thick (almost opaque)
  • This changes how much of the background (such as your wallpaper) is visible behind the text
  • Use a thicker material for higher contrast and legibility
  • Use a thinner material for a more transparent, modern aesthetic

An interactive preview lets you see how each option looks in real-time.
